## Getting Started

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/idaxon/PyLibHub.git
    ```
2. **Install Dependencies**:
    ```bash
    npm install
    ```
3. **Run Locally**:
    ```bash
    npm start
    ```
